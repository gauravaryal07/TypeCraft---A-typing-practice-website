# ⌨️ TypeCraft

> Master your typing speed with a beautiful, distraction-free experience.

TypeCraft is a modern typing practice web application designed to make improving your typing speed enjoyable and engaging. It combines elegant UI design, smooth animations, detailed performance analytics, customizable themes, ambient music, and accessibility features into one seamless experience.

---

## ✨ Features

### 🎯 Typing Practice
- Real-time WPM calculation
- Accuracy tracking
- Error highlighting
- Live progress indicator
- Countdown timer
- Multiple typing modes
- Instant restart

## Installation

Clone the repository

```bash
git clone https://github.com/yourusername/typecraft.git
```

Navigate to the project

```bash
cd typecraft
```

Open the project

Simply open:

```text
index.html
```

No installation or build process is required.

## Why TypeCraft?

Unlike traditional typing websites, TypeCraft focuses on creating a calm, immersive, and highly customizable typing experience. It blends modern UI design, productivity, accessibility, and detailed performance insights into one elegant application.
